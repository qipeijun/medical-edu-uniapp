# 医疗教育小程序 产品&UI设计方案
**设计风格**：科技清新风（主色调：淡青蓝 `#E0F7FA` + 科技蓝 `#03A9F4`，辅助色：浅草绿 `#81C784`、暖白 `#FAFAFA`；字体：思源黑体，简洁易读；视觉元素：轻量线条、低饱和度渐变、圆角卡片）

## 一、产品核心定位
面向医学相关专业学生，以**习题训练+知识复习+科普学习**为核心，结合遗忘曲线的复习推送机制，打造轻量化、高效的医学备考学习工具。

## 二、产品架构设计
### （一）用户小程序端 架构图
医疗教育小程序

└─ 公共组件层：搜索、分类切换、答题组件、笔记组件、分享组件

## 三、用户小程序端 详细设计
### （一）模式选择页（首次进入）
#### 1. 产品逻辑
- 页面默认展示**注册**和**游客浏览**两个主按钮，中间提示文案：`游客浏览仅可查看大众科普板块`
- 游客点击其他功能（如习题、复习），弹窗引导注册，弹窗含`立即注册`和`暂不注册`按钮

#### 2. UI设计（科技清新风）
- **布局**：居中对称布局，暖白背景，顶部放置小程序logo（蓝绿色渐变+十字医疗元素）
- **按钮**：
  - 注册按钮：科技蓝渐变填充，圆角8px，hover时亮度提升10%
  - 游客浏览按钮：白色描边，科技蓝边框，圆角8px
- **提示文案**：浅灰色14px字体，位于按钮中间，下方加一条淡青蓝分割线
- **交互**：按钮点击有轻微缩放动画（0.98倍），弹窗弹出为淡入效果

### （二）注册登录页
#### 1. 产品逻辑
| 登录方式 | 流程说明 |
| ---- | ---- |
| 手机注册 | 输入手机号→获取验证码→设置密码→确认密码→提交→进入信息完善页 |
| 微信授权登录 | 点击微信授权→绑定手机号（可替换）→直接登录→未完善信息则跳转完善页 |
| 账号登录 | 输入手机号/邮箱→输入密码→登录→未完善信息则跳转完善页 |

#### 2. UI设计
- **布局**：表单式垂直布局，输入框上下间距16px，顶部预留返回按钮
- **输入框**：圆角6px，边框为淡青蓝，聚焦时边框加粗为科技蓝，输入内容左对齐15px
- **验证码按钮**：科技蓝背景，白色字体，点击后变为`倒计时60s`，不可点击时为浅灰色
- **微信登录入口**：底部独立区域，微信绿色图标+`微信快捷登录`文字，点击有震动反馈
- **视觉细节**：输入框内添加轻量图标（手机→短信→锁），提升科技感

### （三）信息完善页
#### 1. 产品逻辑
- 必填项：头像、昵称、学校、姓名、学号、年级、专业、班级
- 提交后弹窗提示：`提交成功！信息不可修改，如需修改请凭学生证/录取通知书联系客服`
- 触发场景：未完善信息用户点击“做题”功能时，强制跳转此页

#### 2. UI设计
- **布局**：分区块布局（基础信息区+学籍信息区），区块间用淡青蓝分割线分隔
- **头像上传**：圆形裁剪框，默认占位图为科技蓝渐变头像，点击可选择相册/拍照
- **选择器**：学校/年级/专业/班级使用下拉选择器，展开时为圆角卡片，选项hover为淡青蓝背景
- **提交按钮**：底部通栏科技蓝渐变按钮，文字加粗，点击后有加载动画（转圈的十字图标）
- **提示文案**：红色小字标注在必填项右侧，提交前校验，未填项边框变红并提示

### （四）首页
#### 1. 产品逻辑
| 模块 | 功能说明 |
| ---- | ---- |
| 轮播推荐 | 3-5张轮播图，自动切换（3s/张），点击跳转小程序内链接/公众号文章 |
| 今日复习 | 顶部进度条（总题数/未复习数）→学科列表（显示需复习题数+进度）→点击`开始复习`跳转对应题库 |
| 功能入口 | 课程/习题/考试/收藏/笔记本/商城（开发中模块置灰，点击提示`开发中，敬请期待`） |
| 大众科普 | 分类标签（胶囊式）→科普合集列表（图文卡片）→点击进入合集详情 |

#### 2. UI设计（核心科技清新风展示页）
- **顶部轮播**：全屏宽度，高度160px，圆角8px，图片叠加渐变蒙版（下深上浅），文字白色居中
- **今日复习模块**：
  - 头部：科技蓝背景，白色进度条+`今日需复习：XX题`文字，右侧`全部>`按钮
  - 学科列表：卡片式布局，每卡片显示学科图标（如解剖→骨骼图标）、学科名、`待复习XX题`、`开始`按钮，卡片hover时上浮2px，加阴影
- **功能入口**：2行3列图标布局，图标为科技蓝线性图标，开发中模块图标置灰，文字为浅灰色
- **大众科普模块**：
  - 分类标签：胶囊式，选中为科技蓝背景，未选中为白色描边
  - 合集卡片：左侧图片（圆角6px），右侧标题+发布时间，卡片间距12px，背景暖白

### （五）学习页-习题模块（核心功能）
#### 1. 产品逻辑
1.  一级分类选择（学科/器官）→二级分类列表→选择题库
2.  题库详情页：
    - 顶部：分类层级面包屑+题库标题+`分类切换`按钮
    - 搜索框：可搜索题库内题目
    - 模式切换：做题模式（默认，未做题置顶）/顺序模式（1-n排序）
    - 题型切换：胶囊选项卡（A型题/X型题/名词解释/简答题/论述题）
3.  答题页：
    - 显示：题号/总题数/题型+题目内容+选项/输入框+解析按钮+上下题按钮
    - 交互：选择题即时判对错（对→绿底√，错→红底×）；主观题用户自评`会/不会`，点击解析可折叠查看（图文/视频/音频）
    - 完成后：自动加入复习题集

#### 2. UI设计
- **分类页**：左侧一级分类导航（固定宽度），右侧二级分类卡片，卡片封面图为医学相关图片（如心脏→器官分类），hover时加科技蓝边框
- **题库详情页**：
  - 面包屑：浅灰色文字，分隔符为`>`，点击可返回上一级
  - 模式切换：toggle开关，做题模式→蓝色开启，顺序模式→灰色关闭
  - 题型选项卡：胶囊式，选中为科技蓝，未选中为淡青蓝描边
- **答题页**：
  - 题目区：白色背景，题目文字加粗，选项为圆角单选/多选框，选中为科技蓝填充
  - 解析按钮：浅灰色文字+下拉箭头，点击后展开解析区（淡青蓝背景，内可嵌入视频播放器/音频播放条）
  - 操作按钮：底部固定栏，上一题/下一题按钮为科技蓝，完成按钮为绿色渐变

### （六）复习页
#### 1. 产品逻辑
- 顶部进度条：总复习题数/已复习数
- 复习列表：按学科分类，显示待复习题数+进度，包含速刷判断题入口
- 复习推送逻辑：基于遗忘曲线+用户做题时间+错题次数，后台可配置推送参数，错题推送频率高于普通题

#### 2. UI设计
- 头部：与首页今日复习模块一致，保持视觉统一
- 复习列表：卡片式布局，增加`速刷入口`标签（浅草绿背景），点击后跳转知识点速刷页
- 进度展示：环形进度条（科技蓝填充），显示在学科卡片左侧，直观展示复习进度

### （七）我的页
#### 1. 产品逻辑
| 模块 | 功能说明 |
| ---- | ---- |
| 基础信息 | 显示头像/昵称/学籍信息，点击修改提示`不可修改，联系客服` |
| 我的笔记本 | 搜索+筛选（学科）→笔记列表→点击查看详情（关联题目+笔记内容+分享） |
| 我的收藏 | 习题收藏列表，可左滑删除收藏 |
| 消息通知 | 显示平台推送的复习提醒等消息，未读消息标红 |
| 基础设置 | 关于我们/联系我们（跳转公众号）/在线咨询/意见反馈/隐私政策 |

#### 2. UI设计
- **布局**：个人信息区（顶部通栏）+功能列表区（垂直卡片）
- **个人信息区**：科技蓝渐变背景，头像悬浮在背景上，学籍信息为白色小字，右对齐`> `图标
- **功能列表**：每行一个功能，左侧线性图标，右侧箭头，hover时背景为淡青蓝，开发中模块置灰
- **笔记详情页**：题目内容灰色显示，笔记内容白色背景，分享按钮为科技蓝，点击生成带水印的分享图

## 四、后台管理PC端 详细设计
### （一）通用设计规范
- **布局**：左侧侧边栏（功能导航）+顶部导航栏（账号/退出/消息）+右侧内容区
- **风格**：科技清新风适配PC端，主色调科技蓝，背景为暖白，卡片圆角6px，阴影轻量
- **交互**：侧边栏菜单展开/收起动画，内容区tab切换无刷新，按钮点击有反馈

### （二）核心模块设计
#### 1. 题库管理-题目管理
- **产品逻辑**：支持创建A型题/X型题/名词解释/简答题/论述题，解析可插入图文/视频/音频，支持上下标（如CO₂）
- **UI设计**：
  - 富文本编辑器：支持字体粗细/颜色/上下标，插入按钮为科技蓝图标
  - 选项配置：A型题/X型题直接生成5个选项输入框，支持单选/多选标记
  - 预览功能：编辑后可点击`预览`查看前端显示效果，弹窗为手机模拟器样式

#### 2. 复习管理-参数配置
- **产品逻辑**：配置遗忘曲线参数（首次推送时间/错题推送频次/移出阈值）
- **UI设计**：
  - 滑块配置：推送时间/频次使用滑块组件，实时显示数值，支持手动输入
  - 阈值配置：输入框+单位（次），提交后有`保存成功`提示（绿色弹窗）

#### 3. 数据管理-数据分析
- **产品逻辑**：实时概况（今日/昨日收入对比、新增用户、答题用户）+运营视窗（访客/做题/复习人数）
- **UI设计**：
  - 图表展示：使用柱状图/折线图，配色为科技蓝/浅草绿，支持日期筛选
  - 数据卡片：关键指标（新增用户数）使用大字体展示，背景为淡青蓝渐变，提升视觉层级

## 五、科技清新风 设计要点总结
1. **色彩体系**：以淡青蓝、科技蓝为主色调，浅草绿为辅助色，暖白为背景色，避免高饱和色，营造清新感
2. **视觉元素**：大量使用线性图标、圆角卡片、轻量阴影、渐变蒙版，提升科技感
3. **交互体验**：按钮点击缩放、弹窗淡入淡出、滑块实时反馈，操作流畅无卡顿
4. **医学属性**：融入十字、骨骼、器官等轻量医学图标，强化产品定位
